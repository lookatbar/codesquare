## 通讯录



REQUEST (请求):

|  请求方式    |    POST     |
| :------- | ----: | :---:  |
| 请求地址 | {host}/codesquare/user/address-list   |

请求参数说明：

|    参数名称  | 参数类型 | 是否必须   | 描述   |备注  |
| :------- | ----: | ---: | ---:   | ---:   | 
| token | string |  是    |  令牌  ||
| page_size | int |  否    |  分页大小 ,默认 10 | |
| page_index | int |  是    | 页码，默认为1   |  | 





Response(响应)
```
   {
       "success": 0,
       "message": "ok",
       "data": [
           {
               "user_id": "2",
               "name": "测试"
           },
           {
               "user_id": "baiyy",
               "name": "白燕燕"
           }
        ]
   }
```

响应参数说明：

    | 参数名称  | 参数类型 |  描述   |备注 |
    |:------- | ----:|  ---: |---:   |---:   |
    |data      | array |方式列表||
    |--user_id|int|用户id||
    |--name|string| 用户名称||
