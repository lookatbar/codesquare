|  请求方式    |    POST     |
| :------- | ----: | :---:  |
| 请求地址 | {host}/codesquare/topic/topic-detail  |

请求参数说明：

|    参数名称  | 参数类型 | 是否必须   | 描述   |备注|
| :------- | ----: | ---: | ---:   | ---:   |
| token | string |  是    |  令牌  ||
|||||
|topic_id|int|是|话题ID||

响应参数说明：

|    参数名称  | 参数类型 |  描述  |   备注 |

| data | object |    令牌  |    |
| --topic_id|int|话题ID||
| --title|string|话题标题||
| --topic_type|int|话题标签类型|[1:技术，2:点子]|
| --topic_type_name|string|话题标签名字||
| --content|string|话题内容||
| --images_list|string|图片列表，json 字符串||
| --user_id|int|话题作者ID||

| --user_avatar|string|话题作者头像||
